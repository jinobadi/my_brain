# 🏗️ 지노바디AI COMPANY — MVP 비즈니스 블루프린트 (v1.0)
## 1. MVP 컨셉 & 타겟
- **제품명 가제:** FocusFlow AI
- **타겟:** 프리랜서, 크리에이터, 스타트업 PM 등 '생산적 작업'을 일상화하는 개인
- **핵심 가치:** 로그인 없이 1분 내 작업 초안 생성 → 반복 업무 자동화 → 데이터 프라이버시 보장
- **진입 전략:** Niche-First (특정 직군/워크플로우에 최적화) → 커뮤니티 기반 Viral Loop

## 2. Must-have 기능 (개발 우선순위)
| 우선순위 | 기능명 | 비즈니스 목적 | 구현 복잡도 |
|:---:|:---|:---|:---:|
| P0 | **Instant Value Demo** (로그인/결제 전 사용) | 온보딩 이탈 방지, 신뢰 형성 | ⭐ |
| P0 | **AI 템플릿 실행 엔진** (1회 클릭 자동화) | 반복 작업 제거, 구독 전환 유도 | ⭐⭐ |
| P1 | **로컬-프라이버시 모드** (IndexedDB/LocalStorage) | 클라우드 의존 피로감 해소, 차별화 | ⭐⭐ |
| P1 | **워크플로우 저장/공유** (템플릿 마켓플레이스 기반) | 네트워크 효과, 재방문 유도 | ⭐⭐⭐ |

## 3. 수익화 모델 & 가격 비교 (2026.05 기준)
| 옵션 | 모델 | 가격대 | 타겟 | 전환 예상 | 비고 |
|:---|:---|:---|:---|:---:|:---|
| **안 A (메인)** | 마이크로 구독 | 월 $9 / 연 $79 (20% 할) | 개인 프로듀서 | 3~5% | 진입 장벽 최소화. 월 100회 AI 실행권 포함 |
| **안 B (번들)** | 프리미엄 구독 | 월 $19 / 연 $149 | 팀/고강도 사용자 | 1~2% | 무제한 실행권 + 고급 템플릿 5개 + API 연동 |
| **안 C (일회성)** | 크레딧 팩 | $29 (500회) | 시험형 유저 | 8~10% | 구독 망설임 해소용. 만료일 무제한 (재구매 유도) |

📌 **권고:** 초기 CTA는 **안 A** 고정. 안 B는 3개월 후 '프로페셔널 기능' 오픈 시 병행. 안 C는 Product Hunt/Indie Hackers 공개 시 한정 제공.

## 4. 필수 기술 스택 (1인 개발 최적화)
- **Frontend:** Next.js 14 (App Router) + TailwindCSS + shadcn/ui (UI 개발 시간 70% 절감)
- **Backend/DB:** Supabase (Auth, PostgreSQL, Edge Functions) — 인프라 관리 제로
- **AI 연동:** OpenRouter / Together.ai (API 호출형, 비용 대비 성능 최적화)
- **결제/구독:** Lemon Squeezy 또는 Paddle (글로벌 카드/부가세 대행, PayPal은 키 설정 완료 후 병행)
- **분석/모니터링:** Vercel Analytics + Plausible (과금 없는 프라이버시 파이어월)

## 5. 개발 난이도 vs ROI 분석
- **개발 난이도:** ⭐⭐ (초중반). BaaS + 컴포넌트 라이브러리 + 템플릿 기반 엔진으로 백엔드 개발 최소화.
- **예상 개발 기간:** 2~3주 (기능 확정 시)
- **ROI 예측:** CAC < $5 (콘텐츠/커뮤니티 유통), LTV > $30 (3개월 유지 시). 구독 전환율 3% 달성 시 월 $1,500+ 수익 안정화 가능.
- **핵심 성공 지표 (KPI):** 
  1. P0 기능 사용률 (Day 1 Retention) > 65%
  2. 유료 전환까지 걸리는 시간 < 4분
  3. 구독 3개월 이탈률 < 25%

### 📐 구조 설계 및 수익화 전략 요약

1. **MVP는 '기능 모음'이 아닌 '작업 종료'를 팔아야 합니다.**  
   Researcher의 분석과 일치하게, `로그인 → 결제` 장벽을 제거하고 방문 즉시 AI 초안 생성(P0)을 제공해야 합니다. 1인 개발자는 인프라 관리보다 '전환 경로 최적화'에 리소스를 집중해야 합니다.

2. **수익화 모델은 `안 A (월 $9)`를 메인 CTA로 고정합니다.**  
   마이크로 구독은 초소형 SaaS에서 이탈률과 LTV의 균형점이 가장 명확합니다. 안 B(번들)는 초기에 오픈하면 '안 A가 부족하다'는 인식만 줄 수 있으므로, 3개월 후 타겟 확장 시 병행합니다. 안 C(크레딧)는 유입 확장을 위한 저변 확장형 옵션입니다.

3. **기술 스택은 `BaaS + 컴포넌트 라이브러리`로 개발 시간을 50% 이상 단축합니다.**  
   Next.js + Supabase + shadcn/ui 조합은 1인 개발자에게 가장 높은 ROI를 보장합니다. 결제는 PayPal 키 설정 지연을 피하기 위해 초기에 `Lemon Squeezy` 또는 `Paddle`을 권장합니다. (PayPal은 키 발급 완료 후 병행 연동)

4. **ROI가 높은 방향성: `템플릿 공유 → 네트워크 효과`**  
   사용자가 생성한 워크플로우를 비공개 저장하는 것을 넘어, '템플릿 마켓플레이스' 기능을 P1로 설정합니다. 무료 공유 시 광고 노출 또는 크레딧 보상을 주면, CAC 없이 유기적 유입이 발생합니다.
