# 📐 Next.js Landing Kit (6섹션) 매핑 가이드 & 카피 최종안
**핵심 가치:** `데이터 유출 제로 (Zero Data Leakage)` | **타겟:** 생산적 활동자 (개발자/크리에이터/마케터/1인 기업)

## 🔹 1. 훅(Hook) & CTA 카피 다듬기

### 🎯 훅(Hook) 3안 (A/B 테스트용)
| 유형 | H1 (메인 훅) | Subhead (보조 설명) |
|:---|:---|:---|
| **보안 집중형** | `당신의 아이디어는 클라우드를 지나지 않습니다.` | AI의 속도는 그대로. 데이터는 단 한 바이트도 밖으로 나가지 않습니다. 로컬 WASM 엔진이 생산성과 보안을 동시에 독점합니다. |
| **생산성 집중형** | `클라우드 대기 시간 없이, AI와 함께 일하세요.` | API 호출·업로드·검수 과정이 사라집니다. 브라우저 안에서 바로 실행되는 제로트랜스퍼 AI 엔진을 경험하세요. |
| **주권 강조형** | `데이터는 당신의 것입니다. AI는 당신의 도구입니다.` | 제3의 서버에 정보를 맡기지 마세요. 로컬 추론 엔진이 당신의 업무 데이터를 단 한 번도 외부로 전송하지 않습니다. |

### 🚀 CTA 카피 3안
| 위치 | 카피 | 서브텍스트 (위험 감소) |
|:---|:---|:---|
| **메인 CTA** | `데이터가 사라지지 않는 AI를 무료로 체험하세요` | `신용카드 불필요 · 설치 없이 브라우저에서 즉시 실행` |
| **2차 CTA** | `보안 검수를 기다리는 대신, 바로 시작하세요` | `무료 체험으로 데이터 주권을 되찾으세요` |
| **가격 하단** | `로컬 AI 구독을 시작하기` | `데이터 유출 보험은 모든 요금제에 기본 포함` |

## 🔹 2. 섹션별 매핑 가이드 (Next.js Landing Kit 6섹션 구조)

| 섹션 | 목적 | 핵심 메시지 | 구성 요소 및 카피 지시 |
|:---|:---|:---|:---|
| **1. Hero** | 3초 내 가치 인지<br>신뢰도 형성 | `속도 vs 보안` 이분법 해소 | **H1:** 훅 중 '보안 집중형' 적용<br>**CTA:** 메인 CTA 1안<br>**배지:** `🔒 Zero-Data-Transfer` `⚡ Browser-Native` `🌐 No API Calls` |
| **2. Problem** | 공감 유도<br>대안 부재 인식 | 클라우드 AI의 숨겨진 비용(유출·학습·지연) | **카피:** `클라우드에 올릴 때마다 당신의 정보는 학습 데이터가 됩니다.`<br>**구성:** 3-column 비교 (클라우드 AI vs 로컬 AI). 빨간색으로 `API 요청`, `서버 저장`, `지연 대기` 강조 |
| **3. Solution** | 기술적 신뢰도<br>기능 설명 | 로컬 WASM 작동 원리<br>(네트워크 차단) | **카피:** `데이터는 단 한 바이트도 밖으로 나가지 않습니다.`<br>**구성:** 작동 흐름도 시각화 (Device Memory → WASM Engine → Output). `Network Request: 0` 강조. 기술적 용어는 비전공자도 이해할 수 있게 단순화 |
| **4. Use Cases** | 실제 생산성 증명 | 워크플로우 통합 시너지 | **카피:** `보안 검수를 기다리는 대신, AI와 함께 일하세요.`<br>**구성:** 3-column (개발자/크리에이터/마케터). `Before: Upload → Wait → Edit` vs `After: Type → Generate → Export` 비교 |
| **5. Pricing** | 전환 장벽 낮춤<br>가격 비교 | $9/$19/$29 명확한 가치 대비 | **카피:** `데이터 유출 보험은 가격에 포함되었습니다.`<br>**구성:** 3-tier 카드. 무료 체험 기간 명시. "로컬 엔진 업데이트 무상 제공" 강조. 구독/일회성 선택 토글 |
| **6. Final CTA** | 최종 행동 유도 | 후회 없는 선택 | **카피:** `지금 시작하세요. 데이터는 당신의 것입니다.`<br>**구성:** 메인 CTA 2안 + FAQ 아코디언(유출 우려 해결) + 지원/문의 링크. 배경은 어두운 톤으로 집중도 유도 |
