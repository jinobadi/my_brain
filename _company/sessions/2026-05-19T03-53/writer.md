### 🎯 타겟 페르소나 3가지 (로컬 AI 엔진 기준)

| 페르소나 | 핵심 페인포인트 | 로컬 AI 엔진이 해결하는 가치 |
|:---|:---|:---|
| **1. 민감한 아이디어를 지키는 프리랜서/작가** | 클라우드 서버에 원고·기획안·컨셉이 올라가는 불안감. 이동 중/오프라인 환경에서 AI 도구 사용 불가. | 데이터 외부 전송 제로. 인터넷 끊겨도 즉시 응답. 구독료 없이 영구 사용 가능. |
| **2. 보안 규정의 벽에 막힌 직장인/전문가** | 내부 데이터/고객 정보를 외부 AI에 업로드할 수 없음. IT 보안 정책 위반 리스크. | 온프레미스급 로컬 실행으로 데이터 유출 리스크 제거. 내부망/차단 환경에서도 작동. |
| **3. 반복 업무에 지친 생산적 자영업자/개발자** | 매달 나가는 구독료 대비 제한된 기능. API 호출 한도 관리 및 토큰 비용 부담. | WASM 기반 즉시 연산. 설정 한 번으로 워크플로우 자동화. 무제한 사용 가능. |

### 🪝 랜딩페이지 훅 (문제제시 → 솔루션제시 구조)

**[후크: 시선 집중]**
> 클라우드 AI가 당신의 비즈니스 데이터를 '먹고' 있다는 사실을 알고 계신가요?

**[문제 심화: 공감과 불안 자극]**
민감한 원고, 내부 데이터, 고객 정보가 클라우드 서버를 거치며 발생하는 3가지 치명적 비용:
1. **데이터 유출 리스크** — AI 학습용 데이터로 사용될 가능성
2. **오프라인 작업 불가** — 인터넷 연결이 끊기면 생산성도 함께 중단
3. **점증하는 구독 비용** — 매달 나가는 비용 대비 답은 점점 더 느려지고 있음

**[솔루션 제시: 로컬 AI 엔진의 명확한 가치]**
이제 AI를 '구독'하는 시대를 끝내겠습니다.  
지노바디AI의 로컬 WASM 엔진은 당신의 PC를 직접 서버로 전환합니다.  
데이터는 절대 외부로 전송되지 않으며, 인터넷 없이도 밀리초 단위 응답이 가능합니다.  
보안, 속도, 비용을 한 번에 해결하는 생산성 혁신입니다.

**[핵심 가치 강조: 신뢰 구축]**
`로컬 실행 = 제로 데이터 유출 + 무제한 연산 + 영구 무료(로컬 환경)`  
복잡한 설치 없이 Next.js 기반으로 3분 만에 내 워크플로우에 탑재하세요.

| **Tertiary (기술/설정)** | `로컬 환경 구성 가이드 보기` | 개발자/고급 사용자 타겟. 설치 전 궁금증 해소 |

### 💡 Next.js Landing Kit 연동 가이드
1. **Hero Section**: `[후크]` → `[문제 심화]`를 스크롤 시 페이드인 애니메이션으로 노출. `Primary CTA`를 고정 버튼으로 배치.
2. **Trust/Feature Section**: `[핵심 가치 강조]`를 3열 아이콘+텍스트로 매핑. 로컬 실행 시 시연 GIF/영상 삽입 권장.
3. **Bottom CTA**: `[Secondary CTA]`를 메인 화면 하단에 배치. `Tertiary CTA`는 푸터 또는 FAQ 섹션에 그레이아웃 톤으로 처리.
4. **톤앤매너 적용**: 전체 문장은 `주어+동사` 구조로 간결하게 정리. 기술 용어(WASM/온프레미스)는 페르소나별 문맥에서 자연스럽게 해석되도록 서브카피로 처리.
