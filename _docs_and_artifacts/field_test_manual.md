# 🏆 클럽레이더 2 (Club Radar 2) - 초간단 1분 필드 테스트(Field Test) 사용 설명서

본 문서는 지노바디 대표님께서 현장에서 일반 테스터 및 영업 웨이터들에게 배포하여, **단 1분 만에 핵심 기능을 쉽게 이해하고 테스트할 수 있도록 디자인실장 영자와 정현담 부장이 합작해 제작한 초간단 프리미엄 매뉴얼**입니다.

**"오늘 밤, 내 주변 200m 안의 클럽 손님과 웨이터를 실시간 GPS로 스캔해 매칭하는 3050 실시간 위치기반 매칭 플랫폼"** 입니다.
*   **공식 라이브 테스트 주소**: **`https://club-rader2.onrender.com/`**
*   **테스트 특전**: 테스트 기간 동안 모든 가입 회원(손님/웨이터)에게 **`1000 CR (크레딧)`의 든든한 실탄 포인트가 무상 기본 지급**됩니다!

## 👤 1. [일반 손님 / Player] 1분 이용 가이드 (B2C)

> **"주변 손님을 스캔해 대화하고, 에이스 웨이터에게 바로 부킹 예약을 넣으세요!"**

*   **Step 1. 📡 실시간 레이더 스캔 (지도)**
    *   하단 **[레이더]** 탭을 터치하세요.
    *   내 GPS 위치를 중심으로 **남성 손님은 시원한 네온 블루(🔵)**, **여성 손님은 강렬한 핫핑크(🔴)** 네온 도트로 실시간 스캔되어 반짝입니다.
*   **Step 2. 💬 1초 만에 대화(채팅) 신청**
    *   레이더 지도 아래 목록에서 마음에 드는 상대나 웨이터 카드를 확인한 뒤 **`채팅 💬`** 버튼을 누르세요.
    *   "포인트 차감 컨펌" 확인 창을 누르면 즉시 상대방에게 신청서가 날아갑니다. (상대가 수락하기 전엔 포인트가 절대 차감되지 않습니다!)
*   **Step 3. 😍 네온 이모티콘 수다 & 매칭 성사**
    *   상대방이 수락하면 **[채팅]** 탭에서 영롱한 대화방이 열립니다.
    *   입력창 왼쪽의 **스마일 이모티콘 버튼(😊)**을 눌러 깜찍한 네온 이모지(`🍾 🍻 🔥 💖`)를 섞어 재밌게 수다를 떨며 오늘 밤 약속을 잡으세요!
*   **Step 4. 🍾 담당 웨이터 1초 예약 신청**
    *   하단 **[예약]** 탭에서 실시간 활동 중인 에이스 웨이터들에게 룸/부스 테이블 번호를 선택해 클릭 한 번으로 간편 예약을 신청할 수 있습니다.

## 🍾 2. [영업 웨이터 / Coach] 1분 이용 가이드 (B2B)

> **"반경 내 모든 손님에게 단체 문자를 쏘고, 대화 신청을 수락해 부킹을 유치하세요!"**

*   **Step 1. 📢 VVIP 단체 삐삐 대량 살포 (영업의 정석)**
    *   하단 **[레이더]** 탭 상단의 **`1:1 채팅 활성화`** 영역을 터치해 **[단체 삐삐 발송]**을 켜세요.
    *   홍보 멘트 한 줄만 적고 쏘면, **단 3 CR**의 파격적인 비용으로 내 주변 200m 안의 모든 손님들에게 카카오톡 스타일 푸시 알림으로 **내 영업 삐삐가 1초 만에 일괄 살포**됩니다!
*   **Step 2. 🔔 0.1초 띵동 알림 & 원클릭 즉시 수락**
    *   손님이 나에게 대화를 걸면 청아한 벨소리와 함께 **[채팅]** 탭의 수락 대기방에 카드가 뜹니다.
    *   **`수락 & 연결`** 버튼 하나만 누르면 포인트가 내 주머니로 안전하게 적립되며, 즉각 손님과 1:1 대화방이 활성화됩니다.
*   **Step 3. 📅 VVIP 테이블 예약 승인**
    *   손님들이 나에게 테이블 예약을 신청하면 **[예약]** 탭에 예약 원장이 들어옵니다.
    *   웨이터 전용 대시보드에서 예약 현황을 보고 즉시 수락/거절을 눌러 오늘 밤 고객 유치 현황을 관리하세요!

## 💡 3. 필드 테스트(QA) 진행 시 꿀팁 & 주의사항 (필독)

1.  **💻 브라우저 교차 테스트 시 주의점!**
    *   크롬의 "시크릿 창" 여러 개는 세션이 공유되므로 계정이 꼬입니다.
    *   완벽한 격리 테스트를 하려면 **[크롬 일반창 1개 / 크롬 시크릿창 1개 / Safari나 Firefox 등 다른 브라우저 1개]** 조합으로 열거나, 각자 **개인 스마트폰**을 쥐고 접속하는 것이 가장 완벽합니다.
2.  **🧹 깔끔한 데이터 초기화 (Cascade Clean-up)**
    *   테스트 도중 유저 목록이나 예약 내역을 싹 비우고 처음부터 다시 시작하고 싶다면, 우측 퀵 가입 창 하단의 **`⚠️ QA 데이터 전체 청소`**를 누르세요. 1초 만에 서버 DB가 깨끗이 세탁되어 새출발이 가능합니다.
3.  **🔐 카카오톡형 배려 퇴장 (완벽한 사생활 보호)**
    *   매칭이 끝난 후 채팅방 우측 상단의 **거대한 네온 퇴장(로그아웃 🚪) 버튼**을 누르면 대화방 목록에서 카드가 깨끗이 사라집니다.
    *   참여자 2명이 모두 방을 나가면 백엔드 DB 서버상에서도 대화 내역이 완전히 파괴(Garbage Collection)되어 최고의 보안 성능을 자랑합니다.
